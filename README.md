ueBlog
======


代码clone下来后，在根目录执行命令：npm install，然后执行bower install


打开mongodb服务


在根目录执行命令：node app，程序启动成功！


参与开发的少年们，可以fork然后自己修修改改，有一定成果了pull request请求合并代码！
